Что умеет Chatterbox

🔐

JWT-авторизация

Безопасная авторизация через HTTP-only cookies. Каждый микросервис самостоятельно валидирует access-токен.

⚡

Realtime сообщения

Мгновенная доставка сообщений через WebSocket с помощью Notification Service и RabbitMQ.

👥

Групповые и личные чаты

Поддержка как один-на-один чатов, так и групповых переписок с несколькими участниками.

💾

История сообщений

Надёжное хранение всей переписки в PostgreSQL с возможностью загрузки истории при входе.

📡

Микросервисная архитектура

Три независимых сервиса (User, Chat, Notification) с асинхронной связью через RabbitMQ.

🚀

Масштабируемость и Docker

Каждый сервис контейнеризирован. Легко масштабировать и разворачивать в production.

В результате авторизации в User Service клиент получает пару JWT ключей.
Каждый микросервис самостоятельно валидирует JWT из HTTP-only cookies.
При создании новых сообщений Chat Service публикует событие message.created в RabbitMQ → Notification Service доставляет сообщение по WebSocket клиенту.
